# Assert DataType

Assert that an object is a
[`DataType`](https://r-xla.github.io/tengen/reference/DataType.md).

## Usage

``` r
assert_dtype(x, arg = rlang::caller_arg(x))
```

## Arguments

- x:

  (any)  
  Object to check.

- arg:

  (`character(1)`)  
  Argument name for error messages.

## Value

`NULL` (invisibly). Called for side effects.
