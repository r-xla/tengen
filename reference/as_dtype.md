# Convert to TensorDataType

Convert to TensorDataType.

## Usage

``` r
as_dtype(x)
```

## Arguments

- x:

  (any)  
  Object to convert. Can currently be a string (e.g. `"i1"`, `"i32"`,
  `"f32"`, ...) or a
  [`TensorDataType`](https://r-xla.github.io/tengen/reference/TensorDataType.md)
  object.

## Value

`TensorDataType`
