# Convert to DataType

Convert to
[`DataType`](https://r-xla.github.io/tengen/reference/DataType.md).

## Usage

``` r
as_dtype(x)
```

## Arguments

- x:

  (any)  
  A dtype string (e.g. `"f32"`, `"i64"`, `"bool"`; the aliases `"pred"`
  and `"i1"` map to `"bool"`) or a
  [`DataType`](https://r-xla.github.io/tengen/reference/DataType.md)
  object.

## Value

`DataType`
