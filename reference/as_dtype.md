# Convert to DataType

Convert to DataType.

## Usage

``` r
as_dtype(x)
```

## Arguments

- x:

  (any)  
  Object to convert. Can currently be a string (e.g. `"i1"`, `"i32"`,
  `"f32"`, ...) or a
  [`DataType`](https://r-xla.github.io/tengen/reference/DataType.md)
  object.

## Value

`DataType`
