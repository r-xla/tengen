# Category of a DataType

The category of a
[`DataType`](https://r-xla.github.io/tengen/reference/DataType.md): one
of `"bool"`, `"int"`, `"uint"`, `"float"`, or `"complex"`.

## Usage

``` r
dtype_category(x)
```

## Arguments

- x:

  ([`DataType`](https://r-xla.github.io/tengen/reference/DataType.md))

## Value

`character(1)`

## See also

[`is_dtype_float()`](https://r-xla.github.io/tengen/reference/is_dtype_float.md)
and friends for per-category predicates
